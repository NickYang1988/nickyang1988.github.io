Ubuntu12.04下安装eclipse C/C++开发环境

时间： 2012-12-03   |   分类： 技术文章   程序开发     |   阅读： 415 字 ~1分钟

前段时间在linux上一直使用的是codeblocks，但是感觉提示还是不智能，所以想试试eclipse，下载安装。

安装方法有两种，都很简单。

方法一

打开Ubuntu软件中心，查找到eclipse，然后安装即可

方法二

打开终端

sudo apt-get install eclipse
sudo ap-get install eclipse-pde
sudo ap-get install eclipse-jdt

因为我们需要开发C/C++项目，所以需要额外安装一些插件，CDT。

安装方法：

启动eclipse，选择help -> Install new software… -> work with框中输入：http://download.eclipse.org/releases/galileo,添加完之后会搜索到CDT的一些内容。

选择Collaboration -> Mylyn Bridge:C/C++ Development

选择programming languages -> Eclipse C/C++ Development

然后安装，同意条款后开始安装，大概有几分钟吧，然后会提示是否重启，重启后你会看到welcome页面有C/C++的说明了，建立新项目也有C/C++项目了。

btw.如果你嫌麻烦，那么请去www.eclipse.org下载eclipse IDE for C/C++ 版本安装。
